Intalacion de apache2 en linuxmint

Saludos en este peque√±o posts mostrare como instalar apache2 en nuestra distro linuxmint.
Para ello actualizamos los repositorios y sistema con el siguiente comando:
sudo apt -y upgrade && sudo apt -y upgrade
Esperamos a que finalize y continuamos instalando apache2 con aptitude:
sudo apt -y install apache2
Despues de esto ya tendriamos automaticamente el servicio corriendo en el puerto 80 de nuestro computador. En caso de que no este encendido lo podemos encender con con el siguiente comando:
sudo systemctl enable apache2.service
sudo systemctl start apache2.service
El systemctl enable es para que el servicio se inicie cuando se enciende el computador y el start inicia el servicio. En caso de que quisieramos desactivar el servicio ejecutamos los siguientes comandos:
sudo systemctl disable apache2.service
sudo systemctl stop apache2.service
El systemctl disable es para quitarlo de los servicios que se inician al encender el computador y la opcion stop detiene el servicio por completo. Y bueno espero te fuera de ayuda este posts, hasta la proxima.
