Intalacion de apache2 en linuxmint

Saludos, hoy veremos algunos comandos basicos para linux. A continuacion nombraremos los comandos y sus usos:
comando cd : El comando cd por sus siglas change directory, funciona para entrar en carpetas desde el terminal por ejemplo, digamos que tenemos una carpeta llamada fotos y queremos entrar dentro de ella para eso escribimos
cd fotos
Su uso basico es: cd <nombre de la carpeta>

Comando ls: El comando ls por siglas llamado list, funciona para listar los archivos dentro de una carpeta.
Funciona de la siguiente manera:
user@notebook:~$ ls
Desktop Documents Downloads Music Pictures Public Templates Videos Como podemos ver al hacer el comando ls se listan todas las carpetas dentro del directorio. Asi tambien se listarian si hay archivos de texto etc. Pero no se listan los archivos que estan ocultos, para poder visualizarlos tendremos que agregarle un argumento al comando ls.
user@notebook:~$ ls -a
. Downloads .linuxmint .sudo_as_admin_successful
.. .ecryptfs .local Templates
.bash_history .gnupg .mozilla .themes
.bash_logout .gtkrc-2.0 Music .var
.bashrc .gtkrc-xfce Pictures Videos
.cache .ICEauthority .Private .Xauthority
.config .icons .profile .xsession-errors
Desktop .ipython Public .xsession-errors.old
Documents .irssi .ssh
 Como puedes ver se muestran muchos mas ficheros que estaban ocultos en la carpeta. Tambien podemos visualizar los permisos de los ficheros con el siguiente comando:
ratman@notebook:~$ ls -l

drwxr-xr-x 3 ratman ratman 4096 Feb 7 15:19 Downloads

drwxr-xr-x 2 ratman ratman 4096 Jan 24 18:14 Templates
drwxr-xr-x 2 ratman ratman 4096 Feb 6 09:07 Videos
 Comando mkdir : El comando mkdir de su nombre make directory funciona para crear un directorio o tambien llamado carpeta, su uso es bastante basico:
ratman@notebook:~$ mkdir carpeta
Si hacemos un ls la carpeta se habra agregado:
ratman@notebook:~$ ls -1
carpeta

Documents
Downloads

 comando rm: Este comando funciona tanto para borrar ficheros de texto, como directorios o carpetas, debes tener cuidado en la forma que lo usas con una cuenta de administrador, ya que facilmente puedes borrar alguna carpeta importante del sistema y petar tu computador.
Su uso basico es el siguiente:
ratman@notebook:~$ rm -r carpeta/
ratman@notebook:~$ ls
Desktop Documents Downloads Music Pictures Public Templates Videos

Como podemos ver se borro la carpeta< que creamos anteriormente.
comando mv: comando move, se utiliza para mover archivos o carpetas de una direccion a otra, su uso basico es el siguiente:
mv miarchivo rutadondeseguardara Bueno estos fueron algunos comandos, hare una segunda parte, hasta la proxima.
